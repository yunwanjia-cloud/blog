torch.distributions.categorical.Categorical

CLASS

依据参数 probs 与 logits 来采样.

例子:

参数:

  • prob (Tensor)

    prob 是 $N$ 维向量, 同时采样输出为 $N-1$ 维向量. prob 其中包含的每个 $1$ 维向量为该结果中该位置数据的概率向量.

    注意: prob 必须是非负的, 有限的, 并且和不为 0 . 它会被标准化为和为 1 的形式.

  • logits (Tensor)

    prob 的对数概率版本.

    注意: prob 与 logits 不能同时使用.
