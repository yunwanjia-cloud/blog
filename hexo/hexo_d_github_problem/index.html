hexo d 命令后 github 无法更新问题

发表于2020-04-04|更新于2021-02-07|Hexo
|阅读量:|评论数:

问题

hexo d 后查看仓库, 发现根本没有更新, 仔细观察 shell 的输出, 还会看到如下报错

其中 xxxx 是你的 github 用户名.

解决方案

删除 hexo/.deploy_git 文件, 然后重新尝试 hexo d , 就可以成功更新了.
